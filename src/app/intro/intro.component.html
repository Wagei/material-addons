Project Material-Addons is an addon to Angular Material which is based on the Material Design system.

It contains additional components and patterns which are not in scope of Angular Material and also provides customized themes.

How to Use Material-Addons

Step 0: Install Material

Material-Addons requires an already set-up Angular Material project. To do a fresh start please follow the official Angular Material guide before you continue, but exclude step 4 ("include a theme").

Step 1: Install Material-Addons package

npm install @porscheinformatik/material-addons --save

Step 2: Add the stylesheet at the top of your projects styles.scss

@use 'node_modules/@porscheinformatik/material-addons/themes/poa';

Step 3: Add Material-Addons component to Angular application

To use a component, you need to import the Module of the component in your app.module.ts or in any other module, which needs the component.
