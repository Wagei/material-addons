Properties

Property Type Default Value Description
type string 'info' The type of the alert. Possible values are success, info, warning, and error.
message string '' The message to be displayed in the alert.
size string 'medium' The size of the alert. Possible values are small, medium, and large.
actionText string '' (Optional) The text for the action button. If provided, an action button will be displayed.
closeable boolean true (Optional) If set to false, the alert will not be closeable. Default is true.

Events

Event Description
close Emits when the alert is closed.
action Emits when the action button is clicked.
