Use to display alert messages with different types (success, info, warning, error), sizes (small, medium, large), and optional actions.

Use When

  • Displaying important messages to users
  • Providing feedback after actions
  • Highlighting important information

Behavior

  • Supports multiple alert types: success, info, warning, error
  • Supports multiple sizes: small, medium, large (default is medium)
  • Allows custom actions with an optional action button
  • The alert can be closed using the close icon (default is closeable)

Import the AlertComponent in your app.module.ts and use it via <mad-alert> .

Basic usage with optional values

API Specification

Properties

Property Type Default Value Description
type string 'info' The type of the alert. Possible values are success, info, warning, and error.
message string '' The message to be displayed in the alert.
size string 'medium' The size of the alert. Possible values are small, medium, and large.
actionText string '' (Optional) The text for the action button. If provided, an action button will be displayed.
closeable boolean true (Optional) If set to false, the alert will not be closeable. Default is true.

Events

Event Description
close Emits when the alert is closed.
action Emits when the action button is clicked.
