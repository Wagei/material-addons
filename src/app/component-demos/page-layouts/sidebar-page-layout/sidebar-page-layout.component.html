Following example shows a sidebar page layout with a collapsable sidebar container that can hold different navigation entries.

Design Guidelines

All pages in an application should follow a common layout structure to ensure a consistent user experience across applications.

The Base SidebarLayout includes the following components:

  • Same components as the base page layout
  • plus a collapsable navigation panel on the right hand side
  • the default text color used for the links is #900 from the primary-palette of the theme. As an alternative you can define your own '--palette-color' variable

The Advanced Sidebar Layout (mad-sidebar-layout) integrates a flexible integration system with customizable header, sidebar and content sections.

Design Guidelines

The Advanced Sidebar Layout works with Angular Portals and includes the following components:

  • headerContent - Customizable header content
  • sideBarItems - Navigation items for the sidebar
  • content - Main content area where router components are displayed
  • footer - Footer area

Design Guidelines

  • headerContent - Customizable header content
