Design Guidelines

All pages in an application should follow a common layout structure to ensure a consistent user experience across applications.

The Base SidebarLayout includes the following components:

  • Same components as the base page layout
  • plus a collapsable navigation panel on the right hand side
  • the default text color used for the links is #900 from the primary-palette of the theme. As an alternative you can define your own '--palette-color' variable

The Advanced Sidebar Layout With Custom Header (mad-sidebar-layout) integrates a flexible integration system with customizable header, sidebar, content and footer sections.

Design Guidelines

The Advanced Sidebar Layout With Custom Header works with Angular Portals and includes the following components:

  • headerContent - Customizable header content
  • sideBarItems - Navigation items for the sidebar
  • content - Main content area where router components are displayed
  • footer - Footer area

The Advanced Sidebar Layout With Default Header (mad-sidebar-layout) offers a variation of the sidebar layout that includes a default header with an additional action group.

Design Guidelines

The Advanced Sidebar Layout With Default Header works with Angular Portals and includes the following components:

  • actionGroup - Default header action buttons content
  • title - Default header title
  • hasBackButton - Default header back button appearance
  • content - Main content area where router components are displayed
  • footer - Footer area
